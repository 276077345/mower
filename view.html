启用排序功能

允许或禁止对各个数据列使用排序。 默认情况下，点击每列的表头单元格,可以根据该列对整个表格进行排序。这个选项可以停用这个排序功能。

注意：禁止或者启用单个列的排序功能，可以通过该列的columns.orderableDT选项来实现。 现在的这个ordering选项是一个全局的配置- 如果禁止，整个DataTables都会停用排序功能。默认值：true

综合示例

Datatables有许多方法来获取你的数据，如果你的数据量比较大，这个时候你需要使用服务器模式来处理你的数据。 在服务器模式下，所有的分页，搜索，排序等操作，Datatables都会交给服务器去处理。所以每次绘制Datatables，都会请求一次服务器获取需要的数据。

通过配置serverSideDT 这个属性来打开Datatables的服务器模式，下面的例子展示了Datatables打开服务器模式。

应用分组 应用编号 应用名称 负责人 联系方式 支持单点登陆 支持单点登出 
<table id="list-application" class="table table-striped table-bordered table-hover" width="100%"
 data-serverSide="true" 
 data-paging="true" 
 data-ordering="false" 
 data-ajax="${base}/admin/macula-base/application/apps"
 rel="datatables">
    <thead>
        <tr>
            <th data-name="appGroup">应用分组</th>
            <th data-name="appId">应用编号</th>
            <th data-name="name">应用名称</th>
            <th data-name="supervisor">负责人</th>
            <th data-name="contact">联系方式</th>
            <th data-name="singleSignOn">支持单点登陆</th>
            <th data-name="singleSignOut">支持单点登出</th>
        </tr>
    </thead>
</table>

Html data-* 属性

Datatables是一个高度灵活的表格工具，可以将任何HTML表格添加高级的交互功能。其中有非常多的参数可以配置。但有些参数使用Html data-* 属性配置难于去书写表达，建议高级选项使用Jadascript去激活数据表格。

    </tr>
  </thead>

    <tr>...</tr>
    <tr>...</tr>
    ...
  </tbody>
</table>

配置参数

方法

API方法

Datatables有一个强大的api，用来处理表格上的数据，你可以添加数据到已经存在的表格，或者对已经存在的数据进行操作。API旨在能够很好地操作表格中的数据。API请参考可以参考datatables中文网或者datatables英文网

事件

API方法

Datatables对自己所触发的事件都有监听，我们可以监听这些动作做相应的处理，比如表格重绘的时候，我需要做一些操作

Datatables的事件监听使用 on()DT 方法或者 jQuery.on() ,事件的命名都用dt结尾， 保证和其他的命名不冲突，下面的例子显示了如何监听 draw 事件 

$('#myTable').on('draw.dt',
    function() {
        alert('Table redrawn');
    });

事件请参考可以参考datatables中文网或者datatables英文网

通过JavaScript激活菜单

    $('#selector').vmMenu(options)

参数

可以将选项通过data属性或JavaScript传递。对于data属性，需要将选项名称放到 data- 之后，例如 data-url="" 。

名称 类型 默认值 描述
url string 空 菜单数据加载地址
isAlwaysShown boolean false 是否默认显示二级菜单

事件

事件 描述
clickMenu.mu.vmMenu 当点击菜单功能时，此事件将被立即触发。事件参数里面包含选中菜单及对应的地址。
complete.mu.vmMenu 菜单加载数据源完毕并渲染HTML成功后，此事件将被立即触发。

监听事件使用jQuery的on方法：

$('#myMenu').on('complete.mu.vmMenu', function () {
// do something…
})
